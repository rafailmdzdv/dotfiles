# My personal dotfiles

## Stack

- neovim
- alacritty
- kitty
- tmux
- hyprland

### Another utils

- Rofi (application launcher)
- mpv (videoplayer)
- mc (cli file explorer) and dolphin (gui)
- waybar (status bar)
- wlogout (logout menu)
- sddm (display manager)
